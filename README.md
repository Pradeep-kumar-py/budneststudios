# Budnest Studios

A modern digital atelier website showcasing design and development services. Built with Next.js, React, TypeScript, and Tailwind CSS.

## Overview

Budnest Studios is a professional portfolio website for a digital agency specializing in:
- **Web Development** - Modern, high-performance websites using Next.js and the MERN stack
- **Mobile Apps** - Cross-platform mobile experiences via React Native and Expo
- **Backend Systems** - Secure, scalable server architectures with Node.js and Express
- **AI Integration** - Smart automation and LLM implementation

## Tech Stack

- **Framework**: Next.js 16.2.1
- **UI Library**: React 19.2.4
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Code Quality**: Biome
- **Compiler**: Babel React Compiler

## Getting Started

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Build

```bash
npm run build
npm start
```

### Code Quality

Format and lint code:

```bash
npm run format  # Format with Biome
npm run lint    # Check code quality
```
